起点：
    怎么获取大数据
        爬虫：自动访问多个http网页并抓取公开的数据
        木马：隐私协议、流氓条款、窃取隐私数据
        import requests
        import lxml
        import html5lib
        import os
    怎么存大数据
        大磁盘、云存储、网盘，分布式数据库
        数据仓库
    怎么清洗大数据
        特征工程，去掉垃圾、干扰，留下凭什么知道哪些是有价值的
    怎么看大数据
        可视化工具，描述性统计
    怎么分析大数据
        传统的统计学，优点：能保证绝对正确、科学、可解释
        机器学习处理，缺点：不正确、不科学、说不清楚，优点：无bug无开发周期、正确度最高99.5%(最低0%)、客观性
        import torch
    怎么用大数据
        把分析结果做成散点图、驾驶舱给老板看，以对实践产生能动的反作用
    怎么机器学习
        学习阶段
            传入：(全是数字的没有合并单元格的有至少超过1万行的每1行表示1个独立记录的可以有成千上万列的巨大的)一个Excel文件
            其他传入：可选
            传出：训练后的机器学习的神经网络模型，此模型客观的吸收了超过1万行经验、经验丰富
        使用阶段
            传入：有1行或多行的一个Excel，(列与学习阶段的Excel的列相同)
            传入2：学习阶段传出的模型
            传出：对每1行预测出一个结果，(结果可以直接写在传入的Excel的相应位置)
        评价：通用则无用 万能则无能，正确度低
            综合理论，综合超过1万行经验，如果含大量噪音数据垃圾数据不该综合的也被综合了
            渐进理论，数据越多越接近真相，不要退步、不稳定，停滞、停滞在80%、停滞在无限趋近于99.5%
